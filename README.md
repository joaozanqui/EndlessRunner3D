## O jogador deve desviar dos alimentos prejudiciais à saúde (Cachorro quente, hamburguer e bolo), coletando o máximo de alimentos saudáveis que conseguir (Banana, maçã, cereja e melancia).

### Ao colidir com algum obstáculo o jogador perde o jogo.
